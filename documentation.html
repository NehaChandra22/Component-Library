How to install ?

To unlock the full potential of Catalyst UI and seamlessly integrate its features into your project, follow these simple steps:

1. Add CDN Links 🌐

In your HTML file, include the following CDN links. Place the CSS link in the head tag for styling and the JavaScript link at the end of the body tag for optimal performance.

2. Explore CatalystUI Components 🔍

Once the CDN links are added, dive into the CatalystUI components. Search for the UI elements you need, such as navbar, button, card, etc.

3. Copy HTML Code 📋

For each UI component, find the associated HTML code provided in the documentation. Copy the code snippet.

4. Paste in Your HTML 🎨

Paste the copied HTML code in your project's HTML file where you want to incorporate the specific UI component. Customize as needed to fit your design.

With these simple steps, you'll have Catalyst UI seamlessly integrated into your project, empowering you to create stunning and functional user interfaces.Code triumph! 🌟

Responsiveness 📐

A fantastic way to create responsive designs is by using a style sheet like cui-use.css. CatalystUI simplifies the development of sites that look great on any device!

All UI components from CatalystUI are fully responsive, adapting seamlessly to various screen sizes.
