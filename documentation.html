How to install ?

To unlock the full potential of Catalyst UI and seamlessly integrate its features into your project, follow these simple steps:

1. Add CDN Links 🌐

In your HTML file, include the following CDN links. Place the CSS link in the head tag for styling and the JavaScript link at the end of the body tag for optimal performance.

2. Explore CatalystUI Components 🔍

Once the CDN links are added, dive into the CatalystUI components. Search for the UI elements you need, such as navbar, button, card, etc.

3. Copy HTML Code 📋

For each UI component, find the associated HTML code provided in the documentation. Copy the code snippet.

4. Paste in Your HTML 🎨

Paste the copied HTML code in your project's HTML file where you want to incorporate the specific UI component. Customize as needed to fit your design.

With these simple steps, you'll have Catalyst UI seamlessly integrated into your project, empowering you to create stunning and functional user interfaces.Code triumph! 🌟

Alert 🚨

The Alert component is a versatile and user-friendly feature designed to notify users about critical information, warnings, successes, or errors in your web applications. Whether you want to communicate important messages or enhance user experience, the Alert component offers an intuitive solution.

Alerts are classified into 4 main categories:

  • Success (Green) 🟢
  • Information (Blue) 🔵
  • Warning (Yellow/Orange) 🟠
  • Danger/Error (Red) 🔴

Button

Variants

Catalyst UI includes a variety of button variants, each designed to serve a specific semantic purpose and offer enhanced control over your application's user interface. Use class:'btn' to create button . To create variant buttons use classes given below.

Button Variants:

  • .btn-primary
  • .btn-secondary
  • .btn-success
  • .btn-warning
  • .btn-danger
  • .btn-info

Outline Buttons

To create an outline button, simply replace the default modifier classes with the .btn-outline-* classes. These classes remove all background images and colors, giving your buttons a clean and minimalistic appearance.

Outline Button Variants:

  • .btn-outline-primary
  • .btn-outline-secondary
  • .btn-outline-success
  • .btn-outline-warning
  • .btn-outline-danger
  • .btn-outline-info

Shimmer button

To use Shimmer button put class : 'btn-shimmer'
